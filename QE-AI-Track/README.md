# 🧠 QE-AI-Track — AI for Quality Engineers

**From LLM basics to building autonomous testing agents.**

| ⏱️ Duration | 🎯 Parts | 📊 Difficulty |
|:-----------:|:--------:|:-------------:|
| Self-paced | 21 Parts | Beginner → Advanced |

---

## 🎯 What You'll Learn

By the end of this track, you'll be able to:

- ✅ Understand how LLMs work under the hood
- ✅ Write structured prompts that produce reliable test outputs (RICE-POT)
- ✅ Build AI agents that execute tests autonomously (n8n, LangFlow)
- ✅ Create RAG pipelines that query thousands of test cases instantly
- ✅ Build a Multi-Source QA Copilot (Selenium + Playwright + JIRA + PRDs)
- ✅ Master MCP (Model Context Protocol) for AI tool integration
- ✅ Deploy production-grade multi-agent QA pipelines with CrewAI

---

## 📋 Learning Path

### Part 01 — LLM Basics
- Foundation models, attention mechanisms
- Why prompt wording changes AI output
- Anti-Hallucination Rules & Checklists

### Part 02 — Prompt Engineering
- **RICE-POT Framework** (Role, Instructions, Context, Example, Parameters, Output, Tone)
- STAR, CLEAR, CRISP techniques
- 6 reusable QA prompt templates

### Part 03 — Job Assistant AI
- Build AI tools for daily QA workflows
- Automate repetitive testing tasks

### Part 04 — AI Agents
- Core agent architectures
- Planning, tool use, memory

### Part 05 — AI Agents with n8n
- No-code AI workflow automation
- Resume tailoring, social media agents
- **Human approval gates** before publishing

### Part 06 — AI Agents with LangFlow
- Visual agent building
- Flaky test analyzer
- Vector store integration

### Part 07 — Vibe Coding Agents
- AI-assisted test generation
- Cursor, Claude Code for QA

### Part 08 — RAG (Retrieval Augmented Generation)
- Chunking strategies (300 chars, 50 overlap)
- Embeddings with Nomic + Ollama
- ChromaDB, Qdrant vector stores
- Hybrid search with metadata filters

### Part 09 — Multi-Source QA Copilot ⭐
- **Query 5 sources simultaneously**: Selenium framework, Playwright tests, VWO test cases, PRDs, JIRA bugs
- Stack: FastAPI + React + Qdrant + BGE-M3 + Groq
- Built-in RAG Explorer debugger

### Part 10 — MCP Basics
- Model Context Protocol fundamentals
- Playwright MCP: 23 browser control tools
- AI-driven browser automation

### Part 11 — Python for QA
- Python essentials for testing workflows

### Part 12 — MCP Creation
- Build your own MCP servers
- Custom tool integration

### Part 13-15 — CrewAI (Multi-Agent Systems)
- Single-agent → multi-agent → production pipeline
- Agent collaboration, task delegation
- Production-grade AI QA infrastructure

### Part 16-19 — DeepEval
- AI evaluation frameworks
- Test quality metrics
- LLM output validation

### Part 20 — Browser Automation
- AI-driven browser testing with Browser-pilot

### Part 21 — LangChain
- Advanced LLM orchestration patterns
